# 音声ファイル配置場所

このフォルダに以下のMP3ファイルを配置してください。

- start.mp3
  - 試合開始時に再生されます。
- change.mp3
  - 交代タイミング、または手動交代時に再生されます。
- last-1min.mp3
  - 残り1分で再生されます。
- finish.mp3
  - 試合終了時に再生されます。

例:
public/audio/start.mp3
public/audio/change.mp3
public/audio/last-1min.mp3
public/audio/finish.mp3

MP3ファイル名が一致していれば、ソースコードの変更は不要です。
